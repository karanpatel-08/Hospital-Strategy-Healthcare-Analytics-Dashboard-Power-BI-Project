# 📌 Project Overview

This project focuses on transforming raw healthcare data into strategic hospital intelligence using Power BI.

The objective was to analyze patient behavior, hospital performance, treatment trends, and operational metrics to build an interactive business intelligence dashboard capable of supporting data-driven healthcare decision-making.

The dashboard was developed from a hospital management and healthcare analytics perspective to help stakeholders improve operational efficiency, resource planning, and patient service analysis.

---

# 🎯 Business Problem

Hospitals generate large volumes of operational and patient-related data every day. However, healthcare management teams often face challenges such as:

- Limited visibility into patient trends
- Difficulty monitoring hospital performance metrics
- Inefficient resource allocation
- Lack of centralized operational reporting
- Challenges in identifying high-demand healthcare services

Without proper analytics, hospitals may struggle to optimize operational planning and improve patient service efficiency.

This project aims to address these challenges through interactive healthcare analytics and strategic reporting using Power BI.
